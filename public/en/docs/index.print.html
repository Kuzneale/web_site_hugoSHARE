What Is the Documentation For?

Each wave includes a set of documents: main questionnaires, showcards, drop-off questionnaires, codebooks, and methodology.

  • Main Questionnaires ‚Äì CAPI interviews covering health, social networks, economic status, etc.
  • Drop-off Questionnaires ‚Äì written questionnaires with supplementary questions (national modules, experiments)
  • Showcards ‚Äì cards used during the interviews
  • Codebooks ‚Äì description of variables and categories
  • Methodology ‚Äì information on data collection, imputation, weighting, etc.

For direct links, documents, and materials for each wave, please use the documentation archive linked above.
